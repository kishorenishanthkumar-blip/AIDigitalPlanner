3 Layering — clean separation of concerns

7 Testing strategy

Three categories, each with a clear role. Fixtures keep tests deterministic and offline — no real Anthropic calls during CI.

Unit

tests/unit/
Pure-function tests. No database, no network. Fast — runs in milliseconds.

Integration

tests/integration/
FastAPI TestClient + spun-up Postgres & Redis. Validates wiring end-to-end.

Fixtures

tests/fixtures/
JSON snapshots used as inputs and mocked Anthropic responses — so tests don't burn API credits.
